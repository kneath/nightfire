3. Installation Instructions
============================

1. Unzip Files:

   a. The stats.zip file should be unzipped into a directory on your server. 
      

2. The stats.mdb file needs to be uploaded into a directory where 
   the directory permissions have been set to read/write.


3. Open the data.asp file in your favorite text editor.

   a. Update the "strDB" variable with the correct path to your stats.mdb file.

4. Include the count.asp file in all the pages you wish to collect stats from:
	<!-- #include file="count.asp" -->


That should be it. Any problems please contact us: 
http://www.primalblue.com
support@primalblue.com
